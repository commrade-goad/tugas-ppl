# Tugas PPL

## Note

Required `go 1.24.3`

## Usage

### 1. Install go

__Arch:__
```sh
sudo pacman -S go
```

__Other:__

Figure it yourself

### 2. Build program

```sh
go mod tidy
go build .
```

### 3. Run program

```sh
go run .
```

### 4. Access
can be Accessed at `localhost:3000`.

available Route:

* `/` -> redirect to `/home`
* `/home` -> Home
* `/about` -> About
